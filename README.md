# MoodTrackerrr
About the Project


Mood TrackR is a web app that lets users type how they feel and see a response that includes

A matching background color

A Spotify playlist for that mood

A back button to return to the mood input screen

Personalized greeting with your name



The goal was to make something simple interactive and personal

Creators


Marli and Giana

Parthenon Hackathon 2025

Features
Login with your name

Type any mood input not limited to happy sad or calm

Dynamic background color

Spotify playlist recommendation

Back button to return to mood input

Fully functional in a browser

Challenges
Limited preset moods solved by allowing any typed mood

Missing images or playlists could break the page solved by optional handling in the code

Spotify widgets did not always load fixed with official embed links

Back button did not always switch screens fixed by resetting display states

Organizing files and code HTML CSS and JS was tricky at first

Successes
Made a working interactive app

Learned how to handle dynamic content and user input

Fixed bugs and improved functionality step by step

Hosted it on GitHub Pages for public access

Created something personal and unique
